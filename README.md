# Fantasy-Music

 Fantasy Music is an application similar to that of Fantasy Sports. 
 Instead of drafting a team of athletes and earning points based on their performances, this app allows you to draft artists and track their performance in the music industry. 

## Setup 
1. Clone Project
2. Run `npm start` in the command line
3. Backend server running on localhost:3001
4. Frontend running on localhost:3000
5. Use localhost:3000

## Backend Endpoints

`GET /overall`

`GET /billboard-hot-100-artists`

`GET /billboard-hot-200-albums`

`GET /billboard-hot-100-songs`

`PUT /my_team/update` which will take 2 arguments: artist: string (artist name), my_team: boolean (true if on team)

`DELETE /my_team/delete` whic will take arguemtn artist: string (artist name)

## Frontend Endpoints

`/overall`

`/billboard-artists`

`/billboard-songs`

`/billboard-albums`
